Vad är HTML?

HTML är ett märkspråk som används för att strukturera innehåll på webbsidor. Med HTML kan du skapa rubriker, stycken, länkar och mycket mer.

Grunderna i CSS

CSS används för att styla HTML-element och göra webbsidor visuellt tilltalande. Du kan ändra färger, typsnitt, layout och andra designaspekter.

JavaScript för interaktivitet

Med JavaScript kan du göra dina webbsidor dynamiska och interaktiva. Det är ett kraftfullt programmeringsspråk som används för att skapa allt från enkla animationer till komplexa webbapplikationer.
